Introduction

Les ordinateurs ne disposent pas d’un stockage infini. Et si vous êtes un accro à l’accumulation de fichiers, de photos ou de vidéos, vous savez qu’il vous faut trouver d’autres moyens pour stocker vos données. Il vous faut un support de stockage de données qui allie plusieurs caractéristiques :

Les différents supports de stockage,

  • Le disque dur externe
  • La carte SD
  • La clé USB
  • Le CD/DVD
  • Le Cloud
